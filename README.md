# AgroFarmServ

The GenAI-Powered Agriculture Services Platform is an intelligent analytics and decision-support system designed to help agricultural service providers optimize operations using data and generative AI.

The platform manages and analyzes 10,000+ simulated agricultural service records and uses advanced SQL analytics, secure data pipelines, and Retrieval-Augmented Generation (RAG) to generate actionable insights for agricultural stakeholders.

By combining structured data processing with AI-driven insights, the system enables farmers, agribusinesses, and service providers to make data-informed decisions related to crop services, logistics, and resource allocation.

The system uses a Retrieval-Augmented Generation (RAG) prompt framework to automatically generate secure SQL queries that retrieve and analyze customer support tickets. When a user submits a request (e.g., retrieving the last five support tickets), the system converts the natural language query into an optimized SQL query that joins multiple tables, such as support tickets, user orders, and services.
